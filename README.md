# vue-lottery-pro
- [Demo](https://onesir.github.io/vue-lottery-pro/)

## 项目介绍
基于`lottery-3d`主要进行了如下优化：
- 名单排序每次都随机生成
- 优化界面和交互
- 增加即兴加奖功能
- 增加奖品图片展示功能
- 增加支持动态配置初始奖项功能
- 增加支持中奖作废重新抽奖功能
- 增加背景音乐功能

## 2025年V1.1.1版本更新
- 优化抽奖逻辑，粒度到人设置可抽奖的等级

## 2024年V1.1.0版本更新
- 优化抽奖程序，支持滑动展示奖品和中奖名单，页面更加美观喜庆
- 增加初始化只显示奖项名称，点击后才显示奖品的逻辑
- 强化选择奖项突出效果
- 增加自定义单次抽奖个数功能
- 屏蔽3D球体鼠标滑动事件
- 修复中奖作废重新抽奖的功能bug

## 命令
- 启动: yarn run serve
- 打包: yarn run build
